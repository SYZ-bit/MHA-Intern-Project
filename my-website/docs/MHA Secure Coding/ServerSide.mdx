# Server Side Request Forgery

## Description

### What is an Server Side Request Forgery?

Server-Side Request Forgery (SSRF) attacks occur when an attacker exploits a vulnerability in a web application, allowing them to manipulate a parameter and make requests from the server itself. 
These attacks are commonly used to target internal systems that are shielded from external access by firewalls.
